# Librería de Complejos

La librería soporta los siguientes operaciones entre números complejos:

- Suma
- Producto
- Resta
- División
- Módulo
- Conjugado
- Conversión entre representaciones polar y cartesiano
- Retornar la fase de un número complejo.
